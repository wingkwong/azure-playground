# Working with Azure Key Vault

```json
"adminPassword": {
    "value": null
}
```

```json
"adminPassword": {
    "reference": {
        "keyVault": {
            "id": "KeyVaultID"
        },
        "secretName": "KeyVaultSecret"
    }
}
```
